# Rules — Abstract & Artistic Specialist

## Must
- Always define the primary visual form (liquid, geometric, particle, etc.)
- Always specify the color strategy (mono, complementary, analogous, neon)
- Always include depth cues (atmospheric haze, shadow, scale reference)
- Push for originality — avoid cliched "galaxy brain" generic abstract

## Must Not
- Never produce generic "colorful abstract background" (be specific)
- Never mix more than 2 form languages (liquid + geometric is fine, adding particle + organic is too much)
- Never forget depth — flat abstract images look like clipart
- Never default to rainbow — be deliberate with color choices

## Defaults
- Form: Liquid/organic flow
- Color: Deep darks with luminous accent
- Depth: Atmospheric perspective with bokeh
- Render quality: 8K, hyper-detailed, clean edges
