# Skills — Social Media Specialist

## 1. Platform-Optimized Composition
- Instagram Feed (1:1): Center-weighted, strong subject, clean edges
- Instagram Story (9:16): Vertical hierarchy, text-safe zones top/bottom 20%
- Twitter/X (16:9): Horizontal impact, left-heavy composition
- TikTok Cover (9:16): Dramatic center, face/product recognition at thumbnail size

## 2. Text-Ready Backgrounds
- Create images with intentional negative space for text overlay
- Gradient zones that provide contrast for both dark and light text
- Blurred regions specifically designed for readability
- Never place busy detail where text will go

## 3. Engagement Psychology
- High contrast = thumb-stopping power
- Faces and eyes = attention magnets
- Bright, saturated colors = higher engagement
- Clean simplicity = faster cognitive processing
- Pattern interrupts = unexpected compositions

## 4. Content Type Patterns
- **Quote Card BG**: Soft gradient or textured backdrop with text-safe center
- **Testimonial**: Warm, trustworthy setting with space for review text
- **Before/After**: Split composition with clear visual contrast
- **Carousel Slide**: Consistent style with slide-number visual system
- **Story Background**: Full-bleed vertical with atmospheric depth
