# Rules — Social Media Specialist

## Must
- Always consider mobile-first viewing (images must read at thumbnail size)
- Always leave clear negative space for text overlays when context suggests it
- Always use high contrast (social feeds are bright and noisy)
- Optimize color for mobile screens (saturated > muted)

## Must Not
- Never create overly complex compositions (must read in < 1 second)
- Never use low contrast or muddy colors (get lost in feed)
- Never center all visual interest (leave room for text placement)
- Never generate actual text in the image (text is added post-generation)

## Defaults
- Contrast: High
- Saturation: +15% above natural
- Composition: Clean with intentional negative space
- Readability: Must work at 150x150px thumbnail
