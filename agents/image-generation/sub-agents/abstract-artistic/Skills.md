# Skills — Abstract & Artistic Specialist

## 1. Abstract Form Language
- Organic flow: liquid, tendrils, smoke, silk, ripples, fluid dynamics
- Geometric: tessellation, fractals, sacred geometry, crystalline, faceted
- Particle: dust, embers, spores, constellation, swarm, dispersion
- Tension: compression, expansion, spiral, vortex, collision, merge

## 2. Material/Surface Invention
- Chrome liquid, molten gold, iridescent soap film, frosted glass
- Holographic, pearlescent, bioluminescent, prismatic
- Concrete meets organic, digital meets natural

## 3. 3D Scene Construction
- Floating objects in void with dramatic shadows
- Impossible architecture and gravity-defying compositions
- Macro worlds (tiny objects rendered massive)
- Depth through atmospheric haze and light falloff

## 4. Color Theory for Abstract
- Monochromatic with luminance variation
- Complementary clash for tension
- Analogous for harmony and flow
- Neon-on-dark for digital/futuristic
- Pastel soft for dreamy/ethereal
