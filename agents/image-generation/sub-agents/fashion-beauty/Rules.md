# Rules — Fashion & Beauty Specialist

## Must
- Always describe skin lighting quality (dewy, matte, natural, glass)
- Always specify the dominant color story
- Always include texture language for fabrics/materials
- Reference light quality specifically for beauty shots

## Must Not
- Never use unflattering angles for beauty (harsh underlighting, extreme low angle)
- Never describe unrealistic body proportions
- Never use cold/blue-only lighting on skin (mix with warm fill)
- Never neglect background — it must complement the subject's palette

## Defaults
- Lighting: Soft beauty lighting with editorial edge
- Skin: Natural, healthy, dewy
- Background: Complementary neutral or environmental
- Color: Warm neutral palette with one accent
