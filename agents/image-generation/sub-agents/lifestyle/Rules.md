# Rules — Lifestyle Specialist

## Must
- Always establish time of day through lighting cues
- Always include at least one organic/natural element (plant, wood, stone, linen)
- Always describe the spatial environment (not just the subject)
- Include sensory language (warmth, texture, scent implications)

## Must Not
- Never create sterile or cold environments (unless explicitly requested as "minimal")
- Never overcrowd the scene (max 5-6 objects in frame)
- Never use harsh artificial lighting as default
- Never generate recognizable brand logos

## Defaults
- Time of day: Morning golden hour
- Environment: Modern minimalist home with natural materials
- Color palette: Warm neutrals, earth tones, muted greens
- Mood: Calm, aspirational, inviting
