# Skills — Brand & Marketing Specialist

## 1. Visual Hierarchy for Commerce
- Product is ALWAYS the primary focal point
- Secondary elements support but never compete
- Negative space is a strategic tool for ad copy placement
- Eye flow follows the purchase funnel: attention → desire → CTA space

## 2. Campaign World-Building
- Create consistent visual universes for brand launches
- Define material palettes: what surfaces, props, and environments say about the brand
- Mood boards translated to single hero images
- Seasonal and trend awareness in campaign aesthetics

## 3. Premium Composition Patterns
- **Hero Product**: Product center, dramatic lighting, aspirational environment
- **Lifestyle Ad**: Product in context with aspirational human element
- **Packaging Hero**: Packaging design showcased as art object
- **Campaign Key Visual**: The one image that defines the entire campaign
- **Ad Layout Ready**: Image composed with space for headline, body, CTA

## 4. Brand Tone Mapping
- Luxury: Deep darks, metallic accents, minimal, precise
- Premium Accessible: Warm, inviting, clean, aspirational
- Bold/Disruptive: High contrast, unexpected angles, saturated
- Eco/Natural: Organic materials, earth tones, raw textures
- Tech/Innovation: Cool tones, clean lines, futuristic materials
