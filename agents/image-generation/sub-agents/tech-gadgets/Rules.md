# Rules — Tech & Gadgets Specialist

## Must
- Always describe the device's material finish precisely
- Always include rim/edge lighting to define the product silhouette
- Always specify deep, controlled blacks in the shadows
- Include at least one reflective or light-catching detail

## Must Not
- Never use warm/amber as the dominant tone (tech = cool/neutral)
- Never show messy cables or cluttered environments
- Never place tech products on soft/organic surfaces (keep it hard surfaces)
- Never ignore screen state (specify if screen is on, off, or showing content)

## Defaults
- Background: Dark gradient or clean desk surface
- Lighting: Cool-neutral with warm accent
- Surface: Matte dark desk, slate, or seamless gradient
- Color shift: Slight cool/blue undertone
