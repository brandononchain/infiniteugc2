# Skills — Video Master Agent

## Core Skills

### 1. Content Type Detection
- Parse user requests for video archetype signals
- Distinguish between direct-response (UGC ad) and brand (cinematic) intent
- Detect educational vs. entertainment vs. commercial purpose
- Identify if the request needs a human avatar or can be pure visual

### 2. Script Structure Awareness
- Every video needs: HOOK → BODY → CTA (minimum viable structure)
- Hooks must be 1-3 seconds, pattern-interrupt style
- Body must deliver on the hook's promise
- CTA must match the platform's conversion mechanics

### 3. Provider-Content Matching
- Map content requirements to provider capabilities
- Avatar-required → heygen, hedra_avatar, hedra_omnia
- Motion-required → omnihuman, seedance
- Cinematic-required → sora2pro, veo3
- Budget-aware routing (standard vs. premium vs. mass)

### 4. Multi-Modal Coordination
- Script text + visual direction + audio/music cues
- Caption styling that matches the content's energy
- Text overlay timing and placement
- Transition language (cut, fade, zoom, whip pan)

### 5. Hook Engineering
- Pattern interrupt hooks: "Stop scrolling if..."
- Curiosity gap hooks: "This one trick changed..."
- Social proof hooks: "1 million people tried this..."
- Direct challenge hooks: "You're doing this wrong..."
- POV hooks: "POV: You just discovered..."

### 6. CTA Optimization
- Platform-aware CTA language
- Soft CTA: "Follow for more", "Save this"
- Medium CTA: "Link in bio", "Comment [keyword]"
- Hard CTA: "Shop now", "Use code X", "Limited time"
