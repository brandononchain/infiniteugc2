# Skills — Tutorial & How-To Specialist

## 1. Information Architecture
- Break complex processes into 3-5 digestible steps
- Each step = one clear action + one clear result
- Numbered or sequential labeling for cognitive ease
- Progressive difficulty (easy win first, harder steps later)

## 2. Hook Styles for Education
- "3 things you didn't know about [topic]"
- "Stop [doing X the wrong way]. Here's the right way."
- "In 30 seconds I'll teach you [valuable skill]"
- "The [profession] trick that changes everything"

## 3. Visual Direction for Teaching
- Screen space for text overlay showing step numbers
- Demo area in center frame for showing actions
- Before/after moments within each step
- Zoom-in callouts for important details

## 4. Retention Techniques
- Open loop in the hook (promise all steps, deliver sequentially)
- Mini-payoffs at each step (instant visible result)
- "Bonus tip" surprise near the end
- Save-worthy CTA ("save this for later")

## 5. Script Pacing for Comprehension
- 1 concept per 5-7 seconds
- Pause beat between steps (1 second of breathing room)
- Repeat key terms naturally (repetition = retention)
- End with the full result (payoff the promise)
