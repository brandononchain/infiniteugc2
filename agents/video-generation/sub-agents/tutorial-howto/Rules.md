# Rules — Tutorial & How-To Specialist

## Must
- Always number or label the steps explicitly
- Always deliver on the hook's promise (if you say "3 tips", give 3 tips)
- Always show the result/outcome at the end
- Use clear, jargon-free language
- Include text overlay cues for each step number

## Must Not
- Never skip steps or assume knowledge
- Never exceed 5 main steps (cognitive overload)
- Never use filler that doesn't teach ("so anyway", "moving on")
- Never end without showing the complete result

## Defaults
- Steps: 3 (the magic tutorial number)
- Hook style: "X things you didn't know..."
- CTA: "Save this for later" (bookmark/save)
- Pacing: 5-7 seconds per step
- Tone: Friendly expert, not condescending
