# Skills — UGC Ad Specialist

## 1. Hook Crafting (0-3 seconds)
- Pattern interrupt: Start mid-thought, mid-action, or with a bold claim
- Curiosity gap: Promise a revelation or transformation
- Social proof: Reference popularity or trending status
- Direct address: "You" language, speak directly to viewer

## 2. Problem-Solution Framing
- Agitate the problem in 1-2 sentences (make them feel it)
- Present the product as the discovered solution
- Show the transformation/result
- Make it feel like a personal discovery, not a pitch

## 3. Authenticity Engineering
- Filler words: "like", "honestly", "okay so", "literally"
- Self-interruption: "wait — it gets better"
- Emotional reactions: "I'm obsessed", "I can't stop using this"
- Casual structure: Start in the middle, circle back

## 4. CTA Architecture
- Soft → Hard escalation within the video
- Urgency triggers: limited stock, sale ending, code expiring
- Social proof CTA: "join 10K+ people who already..."
- Action-specific: "tap the link", "comment WANT", "save this"

## 5. Script Structures
- **Classic UGC**: Hook → Problem → Discovery → Demo → Results → CTA
- **GRWM Integration**: Routine context → Natural product mention → Why it works → CTA
- **Unboxing**: Package tease → Opening → First impression → Quick demo → CTA
- **Before/After**: The struggle → Finding the product → The transformation → CTA
